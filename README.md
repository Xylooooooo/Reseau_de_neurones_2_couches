# Réseau de Neurones codé entièrement à la main en Python (appris via Coursera), sans utiliser de frameworks de Deep Learning 

L'objectif est de comprendre les mathématiques derrière l'IA en construisant l'algorithme.

## Fonctionnalités

* **100% NumPy :** Pas de boîtes noires, tout le calcul matriciel est explicite.
* **Architecture Modulable :** Nombre de couches et de neurones paramétrables (ex: `[12288, 30, 1]`).
* **Prédiction :** Test sur des images externes via PIL
